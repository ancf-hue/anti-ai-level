# 📐 Анти ИИ Уровень

> Офлайн-приложение для сравнения углов наклона поверхностей через камеру или датчики телефона.  
> Работает без интернета, без регистрации, без сбора данных.

## 🌟 Особенности

| Функция | Описание |
|---------|----------|
| 📱 **Режим датчиков** | Использует акселерометр и гироскоп смартфона для точного измерения угла |
| 📷 **Режим камеры** | Анализирует изображение через веб-камеру ноутбука/планшета |
| ✍️ **Ручной ввод** | Резервный режим: ввод угла вручную, если нет датчиков и камеры |
| 🌐 **8 языков** | Русский, английский, испанский, немецкий, французский, итальянский, португальский, арабский |
| 🌙☀️ **Темы** | Тёмная и светлая тема с автоматическим сохранением выбора |
| 🔊 **Озвучивание** | Голосовые подсказки на выбранном языке (опционально) |
| 📦 **Офлайн-режим** | Работает без интернета после первой загрузки (Service Worker) |
| 📲 **PWA** | Устанавливается на главный экран как нативное приложение |
| 🔒 **Приватность** | Никакие данные не отправляются на сервер, всё хранится локально |
